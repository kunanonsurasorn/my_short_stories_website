# The Recommended Short Stories Website Project
      โปรเจกต์การพัฒนาเว็บไซต์สำหรับแนะนำผลงานเรื่องสั้นของ mami_rianerva โดยนำเอาความรู้ด้าน HTML และ CSS มาผสมรวมกับทักษะการเล่าเรื่องและการเขียนเรื่องสั้นในนามปากกา mami_rianerva เพื่อพัฒนาทักษะการเขียนเว็บไซต์ https://kunanonsurasorn.github.io/basic_short_stories_website/

# วัตถุประสงค์
      1. เพื่อฝึกทักษะการพัฒนาเว็บไซต์โดยใช้ HTML และ CSS
      2. ต่อยอดทักษะการเขียนเรื่องสั้นและนวนิยายประยุกต์เข้ากับการเขียนโปรแกรมเพื่อสร้างโอกาสใหม่ในด้านการเรียนรู้ด้านการเขียนเว็บเพิ่มเติม
